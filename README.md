# medical-coding
# Medical Coding Lab: LLM Serving with Ollama and Streamlit

## config
- Docker
- Python 3.9+
- Ollama

## Setup
1. Clone the repository
2. Build the Docker image: `./docker-startup build`
3. Run the application: `./docker-startup deploy`

## Features
- Chatbot powered by Ollama
- Streamlit web interface
